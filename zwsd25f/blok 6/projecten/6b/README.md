# Projectweek 6B - CRUD verdieping op 6A

In 6B bouw je verder op je bestaande 6A-project. Je start dus niet opnieuw.

## Korte begrippenlijst

- CRUD (Create, Read, Update, Delete: aanmaken, bekijken, aanpassen, verwijderen)
- Resource route (een routepakket dat standaard CRUD-routes in een keer aanmaakt)
- Controller (bestand met logica tussen route en view)
- Eloquent relatie (koppeling tussen modellen/tabellen)
- Eager loading (relaties vooraf ophalen om veel losse queries te voorkomen)
- Middleware (controlelaag voor routes, bijvoorbeeld inloggen verplicht)
- Gate (centrale rechtencheck op actie)
- Policy (rechtencheck op een specifiek record, bijvoorbeeld een review)
