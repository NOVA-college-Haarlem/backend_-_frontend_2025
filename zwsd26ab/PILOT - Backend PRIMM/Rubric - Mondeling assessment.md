# Rubric — Mondeling assessment Backend (PlayList)

> Beoordeling op **begrip en verificatie**, niet op "werkt de app". Een AI-geholpen app die de student volledig kan verantwoorden = voldoende of hoger. Een werkende app die de student niet kan uitleggen = onvoldoende.

**Schaal per criterium:** Onvoldoende (1) · Voldoende (2) · Goed (3) · Excellent (4)

## Criteria

### 1. Code lezen & uitleggen (weegt zwaar)
| Niveau | Beschrijving |
|---|---|
| Onvoldoende | Kan willekeurig aangewezen regels niet uitleggen; "dat heeft AI gedaan". |
| Voldoende | Legt de meeste regels correct uit; haalt soms begrippen door elkaar. |
| Goed | Legt elke aangewezen regel correct uit, inclusief het *waarom*. |
| Excellent | Legt uit én benoemt alternatieven/gevolgen ("als ik dit weghaal, dan…"). |

### 2. Begrip van de flow (route → controller → model → view)
| Niveau | Beschrijving |
|---|---|
| Onvoldoende | Kan niet aanwijzen wat waar gebeurt. |
| Voldoende | Kan de globale flow benoemen met wat hulp. |
| Goed | Volgt een request zelfstandig en correct door alle lagen. |
| Excellent | Verbindt het met de plain-PHP versie / SQL eronder. |

### 3. Datalaag & CRUD
| Niveau | Beschrijving |
|---|---|
| Onvoldoende | Weet niet hoe data wordt opgehaald/opgeslagen. |
| Voldoende | Legt Read en Create uit; Update/Delete vager. |
| Goed | Legt alle 4 CRUD-operaties correct uit en koppelt aan SQL. |
| Excellent | Legt ook validatie en waarom-keuzes uit. |

### 4. Verificatie & debuggen (de AI-vaardigheid)
| Niveau | Beschrijving |
|---|---|
| Onvoldoende | Kan een geïntroduceerde fout niet vinden of begrijpen. |
| Voldoende | Vindt de fout met hints. |
| Goed | Diagnosticeert de fout zelfstandig en legt de oorzaak uit. |
| Excellent | Lost hem op én voorkomt 'm ("daarom valideer/test je"). |

### 5. Professionele verantwoording
| Niveau | Beschrijving |
|---|---|
| Onvoldoende | Ontwijkt vragen, geen eigenaarschap over de code. |
| Voldoende | Eerlijk over wat zelf/AI is, basishouding aanwezig. |
| Goed | Transparant over AI-gebruik én controle daarop; rustige uitleg. |
| Excellent | Reflecteert kritisch op eigen keuzes en op de AI-output. |

## Uitslag
- **Onvoldoende:** criterium 1 of 2 op niveau 1, óf gemiddeld < 2.
- **Voldoende:** gemiddeld ≥ 2 én criterium 1 ≥ 2 (eigen code kunnen uitleggen is een harde eis).
- **Goed:** gemiddeld ≥ 3.
- **Excellent:** gemiddeld ≥ 3,5 met minstens 3× niveau 4.
