# Userstories

## Medewerker

- US-1: Als medewerker wil ik auto's kunnen categoriseren op type en features
- US-2: Als medewerker wil ik auto's kunnen beheren, status bijhouden, rapporten bekijken
- US-3: Als medewerker wil ik verhuringen afhandelen, auto's inchecken/uitchecken
- US-4: Als medewerker wil ik kunnen zien welke auto's beschikbaar zijn op een bepaalde datum
- US-5: Als medewerker wil ik klanten kunnen beheren: klanten kunnen toevoegen, klanten kunnen wijzigen, klanten kunnen verwijderen en klanten kunnen bekijken
- US-6: Als medewerker wil ik locations kunnen beheren: locations kunnen toevoegen, locations kunnen wijzigen, locations kunnen verwijderen en locations kunnen bekijken

## Onderhoud (medewerker)
- US-7: Als onderhoud wil ik auto's markeren voor onderhoud, onderhoudsgeschiedenis bijhouden
- US-8: Als onderhoud wil ik features kunnen beheren: features kunnen toevoegen, features kunnen wijzigen, features kunnen verwijderen en features kunnen bekijken

## Beheerder (medewerker)

- US-9: Als beheerder wil ik een dashboard kunnen bekijken met statistieken over de auto's, klanten en verhuringen
- US-10: Als beheerder wil ik een rapport kunnen genereren met statistieken over de auto's, klanten en verhuringen
- US-11: Als beheerder wil ik een email kunnen sturen naar klanten met informatie over hun reservering
- US-12: Als beheerder wil ik een email kunnen sturen naar medewerkers met informatie over hun taken

## Klant (klant)

- US-13: Als klant wil ik auto's kunnen reserveren, eigen reserveringen bekijken
- US-14: Als klant wil ik kunnen filteren op features en prijs zodat ik een passende auto vind
