# Userstories

## Manager (medewerker)

- US-1: Als manager wil ik nieuwe gerechten kunnen toevoegen met naam, beschrijving en prijs zodat ons menu actueel blijft
- US-2: Als manager wil ik gerecht informatie kunnen bewerken zodat prijzen en beschrijvingen correct zijn
- US-3: Als manager wil ik gerechten kunnen categoriseren (voorgerecht, hoofdgerecht, dessert) zodat het menu overzichtelijk is
- US-4: Als manager wil ik ingrediënten per gerecht kunnen bijhouden zodat we allergieën kunnen communiceren
- US-5: Als manager wil ik verkooprapporten kunnen bekijken zodat ik weet welke gerechten populair zijn

## Klant (klant)

- US-6: Als klant wil ik het menu kunnen bekijken per categorie zodat ik makkelijk kan kiezen wat ik wil eten
- US-7: Als klant wil ik gerecht details kunnen zien inclusief ingrediënten zodat ik kan checken op allergieën
- US-8: Als klant wil ik gerechten kunnen filteren op prijs zodat ik binnen mijn budget kan kiezen
- US-9: Als klant wil ik een bestelling kunnen plaatsen zodat ik mijn eten kan bestellen
- US-10: Als klant wil ik mijn bestelling geschiedenis kunnen zien zodat ik kan herhalen wat ik lekker vond

## Kok (medewerker)

- US-11: Als kok wil ik nieuwe bestellingen kunnen zien zodat ik weet wat ik moet bereiden
- US-12: Als kok wil ik ingrediënten lijsten kunnen bekijken zodat ik weet wat er in elk gerecht moet        
- US-13: Als kok wil ik bestellingen kunnen markeren als klaar zodat de bediening weet dat het kan worden geserveerd

## Kellner (medewerker)

- US-13: Als kellner wil ik bestellingen kunnen opnemen voor klanten zodat zij kunnen eten
- US-14: Als kellner wil ik kunnen zien welke gerechten klaar zijn zodat ik ze kan serveren
- US-15: Als kellner wil ik allergene informatie snel kunnen opzoeken zodat ik klanten correct kan adviseren

## Beheerder (medewerker)

- US-16: Als beheerder wil ik een dashboard kunnen bekijken met statistieken over de gerechten, klanten en bestellingen
- US-17: Als beheerder wil ik een rapport kunnen genereren met statistieken over de gerechten, klanten en best
