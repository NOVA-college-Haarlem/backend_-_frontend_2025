# Userstories

## Receptiemedewerker

- US-1: Als receptiemedewerker wil ik een nieuwe reservering kunnen aanmaken met kamer, gast en datums zodat ik boekingen kan registreren.
- US-2: Als receptiemedewerker wil ik een reservering kunnen wijzigen (datums, kamer, gast) zodat ik fouten kan corrigeren.
- US-3: Als receptiemedewerker wil ik kunnen zien welke kamers beschikbaar zijn per datum zodat ik de juiste kamer kan aanbieden.
- US-4: Als receptiemedewerker wil ik gasten kunnen in- en uitchecken zodat de kamerstatus up-to-date blijft.

## Gast

- US-5: Als gast wil ik mijn reserveringsdetails kunnen bekijken zodat ik mijn verblijf kan controleren.
- US-6: Als gast wil ik mijn factuur en kosten kunnen inzien zodat ik weet wat ik moet betalen.

## Manager

- US-7: Als manager wil ik de bezettingsgraad per periode kunnen zien zodat ik inzicht heb in drukke en rustige periodes.
- US-8: Als manager wil ik omzetrapporten per kamer en kamertype kunnen bekijken zodat ik kan analyseren welke kamers het best presteren.
